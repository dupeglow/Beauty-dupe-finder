Dupe Finder

Find cheaper cosmetic alternatives based on similar ingredients.

How it works

* User searches for a product (example: “La Mer Cream”)
* The system finds products with similar ingredient profiles
* Shows cheaper alternatives
* Displays price difference
* Includes affiliate links (Trendyol, Hepsiburada)

MVP Scope

* 200–300 cosmetic products
* Ingredient-based matching
* Simple search interface
* Basic ranking (similarity + price)

Example

Input: La Mer Moisturizing Cream
Output:

* Nivea Creme → 90% similar → 95% cheaper
* CeraVe Moisturizing Cream → 85% similar → 80% cheaper

Tech (initial plan)

* Backend: Node.js or Python
* Database: simple JSON or SQLite
* Frontend: basic HTML + JS

Goal

Help users save money by finding high-quality dupes.
